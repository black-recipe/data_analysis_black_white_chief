현재 "3번문제완성본.csv"파일에는 흑백요리사2의 라운드별 요리 정보가 들어있어  
여기서 나는 심사위원인 "백종원","안성재"에 따라서 합격 여부를 예측하는 모델링을 하고싶어.  

여기서 변수들의 설명은 아래와 같아.  
name : 쉐프명  
food : 음식명, "-"은 통편집 된 것으로 정보가 없는 것을 뜻한다.  
round : 몇 round인지  
is_winner : 승리인지 패배인지  
is_allive : 합격인지 불합격인지  
how_cook : 요리가 무슨 기술로 만들었는지  
food_category : 요리 종류  
ingredient : 주된 요리 재료  
temperature : 음식이 따뜻한 것인지 차가운 것인지  
back : 백종원 심사에서 합격을 했는지(합격 1, 불합0)
an : 안성재 심사에서 합격을 했는지(합격 1, 불합 0)
is_back : 백종원이 심사를 담당했는지(담당하면1, 안하면0)
is_an : 안성재가 심사를 담당했는지(담당하면1, 안하면 0, 통편집 당해서 몰라도 0)

만약 흑백요리사3가 나온다면 참가자들은 "어떤 심사위원에게 어떤 요리를 선사하면 합격률이 올라가겠다"라는 것을 알 고 싶어서 예측모델링을 하는 것이고  
1. 안성재가 심사를 했을 경우만 추려내서 모델링
2. 백종원이 심사를 했을 경우만 추려내서 모델링
을 해서 각각의 심사위원에게 통과하기 위해서 어떤 요리를 해야할 지 예측하는 로지스틱 회귀분석을 진행하려고해

너가 이제 코드를 짜주고 분석 결과를 내주면 돼
jupyter notebook 형식으로 짜주고 일단 너가 먼저 분석해서 각 파트마다 결과를 내. 그치만 내가 회귀분석은 그래도 지식이 있으니까 회귀분석 가정이 맞는지 확인하고 그러고 싶어서 그런데 통계학적인 점들도 고려해서 코드를 작성해줘
