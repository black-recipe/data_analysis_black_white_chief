# 흑백요리사 합격 예측 모델 분석 결과 보고서 (Ver. 2)

## 1. 분석 개요
- **목적**: 흑백요리사 데이터셋을 활용하여 심사위원(백종원, 안성재)별 합격 여부를 예측하는 로지스틱 회귀분석 모델을 구축하고, 각 심사위원의 성향에 맞는 필승 전략을 도출함.
- **방법론**:
    - 로지스틱 회귀분석 (Logistic Regression)
    - 다중공선성 진단 (VIF)
    - 잔차 분석 (R-style Residual Plots with LOWESS)
- **주요 변수**: 조리법(`how_cook`), 음식 카테고리(`food_category`), 주재료(`ingrediant`), 온도(`temperature`)

---

## 2. 심사위원별 상세 분석 결과

### (1) 안성재 (Ahn Sung-jae) 심사위원
- **데이터 수**: 139건
- **통계적 유의미성 (p < 0.1)**:
    - **`how_cook_튀김` (p=0.0323)**: 통계적으로 유의미하게 합격 확률을 **낮추는** 요인.
    - **`how_cook_조림` (p=0.0352)**: 통계적으로 유의미하게 합격 확률을 **높이는** 요인.
- **오즈비(Odds Ratio) 분석**:
    - **선호 요인**: 퓨전음식, 중식, 일식 카테고리에서 매우 높은 오즈비를 보임 (단, 모수가 적어 과대추정 가능성 있음).
    - **기피 요인**: 훈제, 삶기, **튀김** 방식의 요리는 합격 확률이 현저히 낮음.
- **해석**: 안성재 심사위원은 재료의 본연의 맛을 살리거나 정성이 들어가는 **조림(Braising)** 방식을 선호하며, **튀김(Deep Frying)** 요리에 대해서는 매우 엄격한 기준을 가지고 있음이 데이터로 확인됨.

### (2) 백종원 (Baek Jong-won) 심사위원
- **데이터 수**: 166건
- **통계적 유의미성 (p < 0.1)**:
    - **`how_cook_조림` (p=0.0514)**: 합격 확률을 높이는 긍정적인 요인.
- **오즈비(Odds Ratio) 분석**:
    - **선호 요인**: **퓨전 조리법**의 오즈비가 매우 높게 나타남. 재료 측면에서는 **곡류, 채소류**를 활용한 요리가 유리함.
    - **기피 요인**: 한식 카테고리의 오즈비가 타 카테고리(중식, 퓨전 등)에 비해 상대적으로 낮게 나타남. 이는 한식에 대해 더 엄격하거나, 다른 장르의 임팩트가 더 컸음을 시사함.
- **해석**: 안성재 심사위원에 비해 다양한 조리법에 관대하며, 특히 **창의적인 시도(퓨전)**나 대중적인 맛(중식 등)을 잘 살린 요리에 높은 점수를 주는 경향이 있음.

## 3. 모델 진단 결과
- **다중공선성 (VIF)**:
    - 두 모델 모두 `food_category`(한식, 중식 등) 변수에서 VIF 값이 30~50 수준으로 높게 나타남.
    - 이는 특정 음식 종류가 특정 조리법이나 재료와 강한 상관관계를 가지기 때문임 (예: 회=일식). 해석에는 주의가 필요하나, 예측 성능 자체에는 큰 문제가 없음.
- **잔차 분석 (Residual Plots)**:
    - 생성된 Jupyter Notebook(`prediction_model.ipynb`)에서 R의 `car::residualPlots`와 유사한 형태의 진단 그래프를 확인할 수 있음.
    - 각 변수별 피어슨 잔차와 LOWESS 곡선을 통해 데이터의 비선형 패터이나 이상치를 시각적으로 검토 가능함.

## 4. 최종 전략 제안 (Conclusion)

| 구분 | 안성재 심사위원 공략 | 백종원 심사위원 공략 |
| :--- | :--- | :--- |
| **추천 전략** | **"정성과 깊은 맛"**<br>심사위원이 선호하는 **조림(Braising)** 기법을 활용하여 식재료의 깊은 맛을 끌어내는 것이 유리함. | **"창의성과 임팩트"**<br>**퓨전** 스타일이나 식감이 살아있는 요리, 혹은 **중식** 스타일의 임팩트 있는 요리가 합격 확률을 높임. |
| **주의 사항** | **"튀김 금지"**<br>데이터상 튀김 요리의 합격률이 매우 저조하므로 피하는 것이 상책임. | **"평범함 탈피"**<br>단순한 한식보다는 차별화 포인트가 있는 메뉴 구성이 필요함. |

> **요약**: 다음 라운드 생존을 위해서는 **안성재에게는 '튀김'을 피하고 정성이 들어간 '조림' 요리를, 백종원에게는 '퓨전' 요소가 가미된 창의적인 요리**를 선보이는 것이 데이터 기반의 최적 전략임.
